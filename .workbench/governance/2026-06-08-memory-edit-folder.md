# 2026-06-08 Memory Edit Folder

## 目标

- 回答记忆是否模型提取、存在哪里、如何分类。
- 候选记忆增加编辑能力。
- 记忆抽屉增加打开记忆文件夹入口。

## 决策

- 当前仍是本地规则生成候选，不是模型智能提取。
- 后续模型提取也必须进入候选队列，不能直接写长期记忆。
- 记忆分类当前保持 MVP：长期记忆和待确认候选。

- 已通过：`npm run build`
