# 知识图谱关系邻域模式

## 原因

用户要求执行 Tolaria 借鉴项 1：在知识图谱里补关系邻域模式，让当前知识卡能快速看到出链、反链和关系分组。

## 变更

- 在知识卡悬浮预览中加入“关系邻域”区块。
- 邻域区块按当前卡片展示关系分组、指向节点和反链节点。
- 复用已有 `edges` 和 `relationships` 数据，不新增后端解析口径。
- 对 frontmatter 双向关系显示“双向”标记。
- 放大图谱预览卡高度，避免邻域信息被裁切。

## 验证

- 已运行 `npm run build`，前端构建通过。
- 已在内置浏览器刷新 `http://127.0.0.1:4180/`，悬浮测试节点后确认显示“关系邻域”、关系分组、反链列表。

## 回退

回退 `src/knowledge/KnowledgeGraphDrawer.tsx`、`src/App.css` 和本治理记录即可恢复到仅显示基础知识卡预览的状态；图谱数据解析未变。
