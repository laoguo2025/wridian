# 2026-06-09 Cocreation Apply Actions

## 背景

继续执行最小 MVP：共创回复已经能显示在右侧面板，但还不能安全落到正文。

## 变更

- 正文编辑器记录当前光标和选区。
- 共创侧边面板增加三个显式动作：
  - 插入到光标处。
  - 替换选区。
  - 复制建议。
- 插入/替换只修改当前编辑器内容，不做自动全文覆盖。
- 真实文件保存继续复用已有自动保存链路。
- 切换文件时默认插入位置重置到文末，避免使用旧文件光标位置。

## 验证

- `npm run build` 通过。
