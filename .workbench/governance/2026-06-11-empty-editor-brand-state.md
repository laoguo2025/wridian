# Empty Editor Brand State

## 背景

用户确认首屏空状态应突出“记忆树、知识库”两大卖点，并指定 slogan 为“让故事有记忆，让知识可调用”，结尾不加句号。Logo 暂时使用 Wridian 字标，下方提供“选择作品库 / 查看记忆树”两个按钮。

## 变更

- 将未选择文件时的稿件编辑区空状态从“文件编辑区”文字占位改为 Wridian 字标、slogan 和两个入口按钮。
- “选择作品库”直接触发作品库文件夹选择。
- “查看记忆树”直接打开创作记忆树抽屉。
- 同步更新项目地图中的首屏空状态长期事实。

## 验证

- `npm run build` 通过。
- 使用本机 Chrome headless 抓取 `http://127.0.0.1:5173` 首屏截图，确认空状态居中显示 Wridian 字标、slogan 和两个入口按钮；纯浏览器环境下右侧对话区出现 Tauri `invoke` 报错，属于非桌面运行限制，不影响本次空状态视觉验证。

## 回退

如该空状态过重或影响写作工作台克制感，可恢复为空文本占位，或仅保留 slogan 与“选择作品库”入口。
