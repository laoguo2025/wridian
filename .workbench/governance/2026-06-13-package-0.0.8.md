# 2026-06-13 Wridian 0.0.8 打包

## 背景

用户要求将 Wridian 升级为 0.0.8，并打包 Windows exe 安装包。

## 变更

- 版本号从 `0.0.7` 升级到 `0.0.8`：
  - `package.json`
  - `package-lock.json`
  - `src-tauri/tauri.conf.json`
  - `src-tauri/Cargo.toml`
  - `src-tauri/Cargo.lock`
- README 增加 0.0.8 更新说明，并将下载入口改为 `release/Wridian-0.0.8-x64-setup.exe`。
- 清理旧 `src-tauri/target/release/resources` 后重新构建，避免历史资源混入安装包。
- 正式 NSIS 安装包已复制到 `release/Wridian-0.0.8-x64-setup.exe`。

## 构建命令

- `npm run build`
- `cmd.exe /d /s /c "\"C:\Program Files (x86)\Microsoft Visual Studio\2022\BuildTools\VC\Auxiliary\Build\vcvars64.bat\" >nul && npm run tauri -- build"`
