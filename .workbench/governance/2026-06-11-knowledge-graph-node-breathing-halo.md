# 知识图谱节点呼吸光晕优化

## 原因

用户反馈知识图谱缩放后，小圆点的光晕仍然偏大，且整体呼吸效果不够明显。

## 变更

- 将节点光晕半径从固定像素加法改为按节点当前像素半径比例计算。
- 节点本体半径加入轻微比例呼吸，避免缩放后固定像素抖动失真。
- 增强光晕透明度和扩张幅度，使呼吸效果在正常视图下更容易感知。
- 保持标签、关系线和关系标签渲染逻辑不变。

## 验证

- 已运行 `npm run build`，前端构建通过。
- 已在内置浏览器刷新 `http://127.0.0.1:4180/`，对比两张间隔截图确认：小节点光晕随圆点尺寸缩放，呼吸变化更明显且未遮挡关系标签。

## 回退

回退 `src/knowledge/KnowledgeGraphDrawer.tsx` 中节点绘制半径和光晕计算即可恢复旧视觉效果。
