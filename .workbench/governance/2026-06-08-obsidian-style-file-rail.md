# 2026-06-08 Obsidian Style File Rail

## 目标

- 左侧文件区借鉴 Obsidian 的结构。
- 顶部提供新建文件和新建文件夹。
- 文件树支持多层显示和节点右键菜单。
- 底部提供选择文件夹和设置入口。

## 边界

- 只借交互结构，不使用 Obsidian 源码。
- 删除动作实现为移动到当前工作根目录 `.wridian-trash/`。
- 后端文件操作放在 `workspace.rs`，`lib.rs` 只注册命令。

- 已通过：`npm run build`
