# 消防救援类短剧专业校正笔记

用于写《烈火雄兵》这类消防、火场、救援职业流短剧时，避免把专业装备写成玄学外挂。

## 热成像仪在火场里的正确戏剧用法

结论：火场里可以使用消防热成像仪，但不能写成“废墟下生命红点定位器”。

更准确的写法：

1. 热成像仪是辅助工具，常用于浓烟环境下提高可见度、搜救、判断火源/蔓延、识别障碍和出口方向。
2. 它看的是表面红外热差，不是 X 光，不能穿透实体废墟直接定位被压人员。
3. 高温均匀环境会让画面白化、饱和、丢失细节；水汽、玻璃、水面、镜面、湿透人员也会造成误判。
4. 不能把温度读数当作精确战术依据；消防热成像仪不是精确温度计，更不能拿来读人体体温。
5. 戏剧中应写成：主角短暂扫视热差，发现画面受高温/水汽/油火干扰，随后靠声音、烟流、触摸、低姿搜索、绳线/水带方向和传统训练判断。

## 推荐镜头写法

- “屏幕先是一片白，随后被水汽糊成灰雾。”
- “高温区域白成一团，门缝下方只残留一块较暗的人形轮廓。”
- “陆燃只扫一眼就把仪器挂回救援包：‘仪器看不清了，听声音。’”
- “热成像仪白化失效，陆燃改用手背贴地探温、听咳嗽声、摸烟流方向。”

## 禁忌写法

- “废墟下出现一个清晰红点。”
- “热成像仪精准显示还有几个人活着。”
- “隔着厚墙/废墟直接看见人体轮廓。”
- “用热成像仪测出人体准确体温并据此判断生死。”

## 剧本执行规则

消防职业流爽点应来自“专业判断 + 训练动作 + 组织调度”，不是装备开挂。装备可以制造视觉锚点，但每次使用都要同步写出限制、误差和主角如何补救。
