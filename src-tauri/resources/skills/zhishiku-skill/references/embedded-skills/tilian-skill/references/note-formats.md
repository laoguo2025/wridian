# 分析笔记格式

## 通用要求

分析笔记属于 `02拆解报告` 的案例产物，不直接进入 `03-07`。

每份笔记必须做到：

- 有 YAML 头。
- 有明确来源。
- 有分析范围。
- 引用原文或描述具体画面。
- 技法命名具体到操作层面。
- 后续知识卡能通过 `编译来源` 回到这份笔记。

## 路径 A/B：事件流型

适用：小说、叙事文本、长篇故事梗概。

```yaml
---
title: 分析笔记-《作品名》-{卷/段标识}
来源: {作品全名} {作者名}
阅读范围: 第X卷-第Y卷（第X章-第Y章）
分析日期: YYYY-MM-DD
核心命题: {1-2句提炼本段核心叙事命题}
---
```

板块标题统一使用：

```markdown
## 一、事件流
## 二、人物轨迹表格
## 三、技法拆解
## 四、待追踪线索表格
```

禁止：

```text
# 一、事件流
## 一、事件流（40%）
## 一、事件流（≥25节点）
## 三、技法拆解（5项）
```

| 板块 | 写法 |
|---|---|
| 一、事件流 | 数字编号。每个事件写核心动作一句话 + 关键原文引用 |
| 二、人物轨迹表格 | 角色 → 初始状态 → 关键转折 → 终结状态 → 核心执念 |
| 三、技法拆解 | 3-5 项。命名 + 操作描述 + 为什么有效 + 可复用性 |
| 四、待追踪线索表格 | 线索 → 当前状态 → 预期走向。标注新埋/延续/已回收/可能是闲笔 |

## 路径 I：视听拆解型

适用：剧本、电影、电视剧、短剧、游戏剧情。

```yaml
---
title: 分析笔记-{片名}-{幕/章节/集数}
来源: {片名}
导演/编剧: {名字，可空}
场景范围: 第X场-第Y场 / 第X集 / 第X章节
分析日期: YYYY-MM-DD
---
```

| 板块 | 写法 |
|---|---|
| 场景流 | 场景 → 核心动作 → 功能 → 视听描述 |
| 视听语言 | 镜头运动、构图、色彩光线、关键道具视觉 |
| 声音设计 | 对白节奏、音效、配乐、静默运用 |
| 技法拆解 | 从视听角度拆，不只复述剧情 |
| 结构与节奏 | 这批场景在整部作品中的位置和节奏 |

英文源处理：原文引用保留原语言，技法拆解和分析用中文。

## 路径 E：说服链拆解型

适用：公众号文章、营销文案、销售页、小红书爆文、短视频口播稿。

```yaml
---
title: 分析笔记-{文章标题}
来源: {公众号/作者/平台}
文章类型: 观点文/故事文/清单文/销售页/其他
分析日期: YYYY-MM-DD
---
```

| 板块 | 写法 |
|---|---|
| 钩子拆解 | 标题和前三句用了什么钩子 |
| 说服链 | 主论点 → 证据 → 分论点 → 证据 → 结论 |
| 情绪操控拆解 | 焦虑、渴望、认同、转化钩子的位置 |
| 可复用技法 | 标题公式、过渡、金句、结尾转化模板 |

## 路径 D：论点链型

适用：方法论书籍、写作教程、理论书、工具书。

```yaml
---
title: 分析笔记-{书名}-第X章
来源: {书名}
作者: {作者名}
章节: 第X章 {章节标题}
分析日期: YYYY-MM-DD
---
```

| 板块 | 写法 |
|---|---|
| 核心论点 | 本章核心主张 |
| 论证链路 | 作者如何论证，用了什么例子 |
| 可提取的方法论 | 转化为步骤、清单、判断标准 |
| 与知识库的对接点 | 对应 `06写作技法` 下的单点技法、组合技法、审美法则，或其他目录 |

## 路径 J：视觉符号拆解型

适用：图片、电影截图、海报、漫画单页、AI 生图参考。

```yaml
---
title: 分析笔记-{图片描述/标题}
来源: {出处}
图片类型: 电影截图/海报/漫画/AI生成/摄影/其他
分析日期: YYYY-MM-DD
---
```

| 板块 | 写法 |
|---|---|
| 画面描述 | 客观描述画面里有什么 |
| 构图与焦点 | 视线如何被引导 |
| 色彩与光 | 主色调、光源、明暗对比、情绪 |
| 符号层 | 道具、站位、空间关系的隐喻 |
| 叙事暗示 | 这张图暗示什么故事或疑问 |
| 可提取元素 | 可进入 `05情节方程`、`06写作技法`、`07综合素材` 的内容 |

## 路径 K：短视频秒级拆解型

适用：抖音、快手、B站短视频、Reels、直播切片。

```yaml
---
title: 分析笔记-{视频标题/描述}
来源: {账号名} {平台}
时长: XX秒
视频类型: 剧情/口播/混剪/Vlog/教程/带货/其他
数据: 点赞X 评论X 转发X（如果能获取）
分析日期: YYYY-MM-DD
---
```

| 板块 | 写法 |
|---|---|
| 秒级事件线 | 0-3 秒、3-7 秒、7-15 秒、15 秒到结束 |
| 钩子机制 | 前 3 秒为什么留人 |
| 情绪曲线 | 情绪起伏和转折点 |
| 视听手段 | BGM、字幕、运镜、转场 |
| 转化机制 | 点赞、关注、购买、转发怎么触发 |
| 可复用模板 | 是否能抽象为流量方程 |
